# Button

버튼은 `min-width`만 정해져 있습니다.

`width`는 `children`의 길이와`size`프로퍼티에 따라 `padding`이 증감하며 조절됩니다.

  <Story name="default" args={{ children: 'default' }}>

  </Story>
</Canvas>
